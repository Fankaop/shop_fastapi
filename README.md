# Интернет-магазин одежды (FastAPI)

Это backend-проект интернет-магазина одежды: каталог товаров, работа с базой данных и API-слой для дальнейшего развития (пользователи, корзина, заказы, фильтрация по категориям/размерам и т.д.).

Сейчас проект представляет стартовую архитектуру на FastAPI с подключением к MySQL через SQLAlchemy.

## Что внутри

- **FastAPI** — API-приложение
- **SQLAlchemy** — ORM и работа с моделями
- **PyMySQL** — драйвер подключения к MySQL
- Базовая структура каталогов для развития проекта интернет-магазина:
  - `api/` — роуты API
  - `models/` — модели БД (товары, категории, пользователи и т.д.)
  - `schemas/` — Pydantic-схемы
  - `repository/` — слой доступа к данным
  - `services/` — бизнес-логика

## Установка

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Настройка базы данных

Текущая строка подключения находится в `app/core/database.py`:

```python
DATABASE_URL = "mysql+pymysql://root:12345@localhost:3306/shop_db"
```

Перед запуском убедитесь, что:

- MySQL запущен
- база `shop_db` создана
- логин/пароль в `DATABASE_URL` корректны

## Запуск

```bash
uvicorn main:app --reload
```

После запуска:

- API: `http://127.0.0.1:8000`
- Swagger: `http://127.0.0.1:8000/docs`

## Идея развития для магазина одежды

В рамках этого проекта можно реализовать:

- каталог одежды (футболки, джинсы, куртки и т.д.)
- атрибуты товара: размер, цвет, бренд, цена, наличие
- корзину и оформление заказа
- избранное
- фильтры и сортировку
- роли: клиент/админ (управление ассортиментом)
